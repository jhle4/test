[1mNAME[0m[m
       wc - print newline, word, and byte counts for each file[m
[m
[1mSYNOPSIS[0m[m
       [1mwc[0m [[4mOPTION[24m]... [[4mFILE[24m]...[m

[1mDESCRIPTION[0m[m
       Print newline, word, and byte counts for each FILE, and a total line if[m
       more than one FILE is specified.  A word is a non-zero-length  sequence[m
       of characters delimited by white space.[m
[m
       With no FILE, or when FILE is -, read standard input.[m
[m
       The  options  below may be used to select which counts are printed, al‚Äê[m
       ways in the following order: newline, word,  character,  byte,  maximum[m
       line length.[m
[m
       [1m-c[0m, [1m--bytes[0m[m
              print the byte counts[m
[m
[7m Manual page wc(1) line 1 (press h for help or q to quit)[27m[K[K       [1m-m[0m, [1m--chars[0m[m
              print the character counts[m
[m
       [1m-l[0m, [1m--lines[0m[m
              print the newline counts[m
[m
       [1m--files0-from[0m=[4mF[24m[m
              read  input  from the files specified by NUL-terminated names in[m
              file F; If F is - then read names from standard input[m
[m
       [1m-L[0m, [1m--max-line-length[0m[m
              print the maximum display width[m
[m
       [1m-w[0m, [1m--words[0m[m
              print the word counts[m

       [1m--help[0m display this help and exit[m
